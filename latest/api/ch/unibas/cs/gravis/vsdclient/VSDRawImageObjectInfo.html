case class VSDRawImageObjectInfo(rawImage: VSDRawImage, id: Int, createdDate: String, name: String, description: Option[String], ontologyCount: Option[Int], type: VSDObjectType, downloadUrl: String, license: Option[VSDURL], files: VSDPaginatedList[VSDURL], linkedObjects: Option[VSDPaginatedList[VSDURL]], linkedObjectRelations: Option[VSDPaginatedList[VSDURL]], ontologyItems: Option[VSDPaginatedList[VSDURL]], ontologyItemRelations: Option[VSDPaginatedList[VSDURL]], objectPreviews: Option[Seq[VSDURL]], objectGroupRights: Option[Seq[VSDURL]], objectUserRights: Option[Seq[VSDURL]], selfUrl: String) extends VSDObjectInfo with Product with Serializable

Information for VSD Objects of type RawImage (i.e. raw intensity images). Additionally to the common object information, this adds optional fields such as slice thickness, inter-spacing, signal strength, modality, .. The sub-fields names should be self-explanatory. In case of doubt, please check the VSD's API doc
